# 🧠 MNIST Digit Classification API (FastAPI + TensorFlow)

## 🚀 Project Overview
This project is a Machine Learning API that classifies handwritten digits (0-9) using a trained Artificial Neural Network (ANN).  
It is built using **FastAPI** and deployed using **Docker / Hugging Face Spaces**.

---

## ⚙️ Tech Stack
- Python
- FastAPI
- TensorFlow / Keras
- NumPy
- Pillow
- Docker

---

## 📌 Features
- Upload handwritten digit image
- Predict digit (0–9)
- Return confidence score
- Fast API response
- Docker-ready deployment
